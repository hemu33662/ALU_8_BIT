# ALU_8_BIT
Designed an 8-bit ALU using the vivado Xilinx software that performs all 8-bit Arithmetic and Logical
operations


Pre-requisites
--------------

- Verilog
## Demo

![Results](ALU_BLOCK.png)

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Logical shift left
- Logical shift right
- Rotate left
- Rotate right
- Logical AND
- Logical OR
- Logical XOR
- Logical NOR
- Logical NAND
- Logical XNOR
- Greater comparison
- Equal comparison
